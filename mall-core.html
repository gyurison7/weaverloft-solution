      기존 브랜드가 보유한 CRM, ERP, 자사몰, POS 데이터와 빠르고 유연하게 연동 구축이 가능하여 고객은 일관된 경험을 할 수 있고 고객 충성도가 증대

    네이티브

  • 프렉티스
    베이스드

  • 옴니채널

    스마트폰, 웹, 오프라인 스토어, 소셜채널 등 다양한 채널에서 동일한 사용자 경험과 구매가 가능한 다양한 채널을 통합 운영이 가능한 커머스 플랫폼

    • 다양한 채널 확장

  • 모듈러

몰코어의 다양한 기능에 대한 타이틀이 들어갑니다.

  • 상품 전시 01

    다양한 상품 구성과 옵션 구성 등으로 편리한 상품 구성 및 관리가 가능합니다.

  • 회원/프로모션 02

    회원 등급, 유형, 차별화된 이벤트 및 프로모션이 가능하며, 자체 멤버십 기능을 기반으로
    고객 로열티 향상 프로그램 운영이 가능합니다.

  • 주문/배송 03

    고객지원 기능을 기본 제공하고 있으며, 별도의 CS 시스템 또는 솔루션과의 연동 개발을 지원합니다.

  • 개인화 추천 마케팅 04

    AI 기반 데이터 분석을 통해서 회원들에게 개인화 추천 마케팅 및 경험을 제공합니다.

    • 개인화 상품 추천 서비스
    • 자동 세그먼트 타겟팅
    • 오프사이트 메시징

  • 검색 05

    커머스 환경에 특화된 검색 기능을 제공합니다.

    • Ai기반 검색 기능
    • 검색 어드민 제공

    ERP, CRM, OMS, POS, PG 외부 연동 API 제공 및 추가 개발 가능
